# 調査目的

本章では、提案システムの設計に先立って実施した予備調査について報告する。予備調査の目的は以下の3点である。

第一に、**既存システムの課題把握**である。市場に存在する猫画像認識アプリケーションを実際に使用し、その機能・性能・ユーザビリティを評価することで、改善すべき点を明らかにする。

第二に、**ユーザーニーズの深掘り**である。猫を飼育している一般ユーザー、動物病院スタッフ、ペット関連サービス事業者へのインタビューを通じて、猫画像認識に対する潜在的なニーズを抽出する。

第三に、**技術的実現可能性の検証**である。予備的なプロトタイプを作成し、提案手法の基本的な有効性を確認するとともに、実装上の課題を洗い出す。

これらの調査結果は、第4章で述べるシステム設計における要求定義の基礎データとなる。
