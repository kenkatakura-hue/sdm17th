# 利害関係者要求（StRS）

利害関係者要求を以下に定義する。

## 利害関係者の同定

1. **エンドユーザー**：ペットカメラ利用者、猫の飼い主
2. **デバイスメーカー**：スマートペットカメラ製造企業
3. **サービス提供者**：ペット見守りサービス運営企業
4. **研究コミュニティ**：画像認識研究者

## 利害関係者要求一覧

| ID | 要求 | 利害関係者 |
|----|------|------------|
| StRS-001 | リアルタイムで猫を認識できること | エンドユーザー |
| StRS-002 | 高い認識精度を有すること | 全員 |
| StRS-003 | エッジデバイスで動作すること | デバイスメーカー |
| StRS-004 | 再現可能な評価が可能であること | 研究者 |
