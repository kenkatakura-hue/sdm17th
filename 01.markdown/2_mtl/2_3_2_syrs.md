# システム要求（SyRS）

利害関係者要求をシステム要求に変換する。

## 機能要求

| ID | トレース | 要求内容 |
|----|----------|----------|
| SyRS-F001 | StRS-001 | 入力画像から猫の有無を判定する |
| SyRS-F002 | StRS-001 | 猫の位置をバウンディングボックスで出力する |
| SyRS-F003 | StRS-002 | 複数の猫を同時に認識する |

## 非機能要求

| ID | トレース | 要求内容 |
|----|----------|----------|
| SyRS-N001 | StRS-001 | 処理速度30FPS以上 |
| SyRS-N002 | StRS-002 | 認識精度90%以上（mAP） |
| SyRS-N003 | StRS-003 | モデルサイズ50MB以下 |
| SyRS-N004 | StRS-003 | 推論時メモリ使用量500MB以下 |
