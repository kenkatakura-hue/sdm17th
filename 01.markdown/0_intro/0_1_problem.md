# 問題意識

深層学習技術の発展により、画像認識の精度は飛躍的に向上している。特にImageNetを用いたベンチマークでは、人間の認識能力を超える性能が報告されている。しかしながら、特定のドメインにおいては依然として課題が残されている。

猫画像の認識は、その代表的な課題領域の一つである。猫は犬と異なり、品種間での形態的差異が小さい一方で、個体内での姿勢変化が大きいという特徴を持つ。また、毛並みのパターンは無限に近い多様性を示し、同一個体であっても照明条件や撮影角度によって大きく異なる外観を呈する。

既存の汎用画像認識モデルでは、これらの猫特有の課題に対して十分な対応ができていない。ResNetやEfficientNetといった高性能モデルを用いても、猫の個体識別においては80%程度の精度に留まることが報告されている。

本研究では、この問題を解決するため、猫の形態学的特徴を明示的に考慮した新規アーキテクチャの開発に取り組む。
