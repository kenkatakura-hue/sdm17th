# 実験目的

評価実験の目的は、提案手法であるNekoNetが、第4章で定義したシステム要求を満たしているかを検証することである。

具体的には、以下の4点を検証する。

1. **検出性能の検証**：Cat Detectorが、mAP@0.5で92%以上の精度を達成しているか（SyR1）
2. **分類性能の検証**：Breed Classifierが、Top-1 Accuracyで85%以上を達成しているか（SyR2）
3. **識別性能の検証**：Individual Identifierが、Rank-1 Accuracyで92%以上を達成しているか（SyR3）
4. **実行性能の検証**：処理速度が30FPS以上（33ms/frame以下）、モデルサイズが10MB以下であるか（SyR4, SyR5）

また、既存手法との比較を通じて、提案手法の優位性を示す。
